jquery-sortScroll

Sorting without moving !

Demo

Click on the "down" arrow to change this paragraph's order in the page.
The element being sorted will stay still while the rest of the page will scroll behind it.

Set-up

<!DOCTYPE html>
<html lang="en-us">
<head>

    <!-- insert css -->
    <link rel="stylesheet" href="http://cdn.rawgit.com/jadus/jquery-sortScroll/master/jquery.sortScroll.css">
</head>
<body>

    <!-- insert jquery.sortScroll.js after jquery core -->
    <script type="text/javascript" src="https://code.jquery.com/jquery-2.1.4.min.js"></script>
    <script type="text/javascript" src="http://cdn.rawgit.com/jadus/jquery-sortScroll/master/jquery.sortScroll.js"></script>
</body>

Basic usage

<!-- automatic initialization using css classes and data attributes-->
<div class="sort-scroll-container">
    <div class="sort-scroll-element" data-animation-duration="1000" data-easing="swing">
        content1
        <button class="sort-scroll-button-up">up</button>
        <button class="sort-scroll-button-down">down</button>
    </div>
    <div class="sort-scroll-element">
        content2
        <button class="sort-scroll-button-up">up</button>
        <button class="sort-scroll-button-down">down</button>
    </div>
    <div class="sort-scroll-element">
        content3
        <button class="sort-scroll-button-up">up</button>
        <button class="sort-scroll-button-down">down</button>
    </div>
</div>

Javascript initialization

//showing default options here
$(".my-container-class").sortScroll({
    animationDuration: 1000,// duration of the animation in ms
    easing: "swing"// easing type for the animation
});

Programmatic sorting

//you can sort without buttons if you want
//here we move the element in slot 3 (zero based) 2 slots up (in slot 1)
$('.my-container-class').sortScroll('sortElement',3,-2)

Events

//you can do something when sorting starts and when it's done
$('.my-container-class').on('sortScroll.sortStart', function(event, element, initialOrder, destinationOrder)
{
    var title = element.find('h2').text();
    console.log("element named "+title+" is being reordered from slot n째"+initialOrder+" to slot n째"+destinationOrder);
}

$('.my-container-class').on('sortScroll.sortEnd', function(event, element, initialOrder, destinationOrder)
{
    var title = element.find('h2').text();
    console.log("element named "+title+" was reordered from slot n째"+initialOrder+" to slot n째"+destinationOrder);
}
