Jquery-sortscroll

Reordering without moving !

View on GitHub Download .zip Download .tar.gz

Usage

<div class="sort-scroll">
    <div class="sort-scroll-element">
        content1
        <button class="sort-scroll-button" data-direction="-1">up</button>
        <button class="sort-scroll-button" data-direction="1">down</button>
    </div>
    <div class="sort-scroll-element">
        content2
        <button class="sort-scroll-button" data-direction="-1">up</button>
        <button class="sort-scroll-button" data-direction="1">down</button>
    </div>
    <div class="sort-scroll-element">
        content3
        <button class="sort-scroll-button" data-direction="-1">up</button>
        <button class="sort-scroll-button" data-direction="1">down</button>
    </div>
</div>
