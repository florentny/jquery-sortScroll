Jquery-sortscroll

Reordering without moving !

View on GitHub Download .zip Download .tar.gz

Demo

Click on the "down" button to change this paragraph's order in the page.
This jquery plugin is specially useful when you want to sort blocks on a page taller than the screen.
The element being sorted will stay still while the rest of the page will scroll behind it.

Simple usage example

<div class="sort-scroll">
    <div class="sort-scroll-element">
        content1
        <button class="sort-scroll-button" data-direction="-1">up</button>
        <button class="sort-scroll-button" data-direction="1">down</button>
    </div>
    <div class="sort-scroll-element">
        content2
        <button class="sort-scroll-button" data-direction="-1">up</button>
        <button class="sort-scroll-button" data-direction="1">down</button>
    </div>
    <div class="sort-scroll-element">
        content3
        <button class="sort-scroll-button" data-direction="-1">up</button>
        <button class="sort-scroll-button" data-direction="1">down</button>
    </div>
</div>

Programatic use

//init plugin (default options below)
$(".my-container-class").sortScroll(
{
    buttonClass: "sort-scroll-button",//css class for sorting buttons
    sortableClass: "sort-scroll-element",//css class for sortable elements
    movingClass: "sort-scroll-moving",//css class applied to the element being sorted
    animationDuration: 1000,// duration of the animation in ms
    easing: "swing"// easing type for the animation
});

//you can do something when sorting is done
$('.my-container-class').on('sortScroll.sortEnd', function(event, element, initialOrder, destinationOrder)
{
    var title = element.find('h2').text();
    console.log("element named "+title+" was moved from slot n°"+initialOrder+" to slot n°"+ destinationOrder);
}

//you don't really need buttons
$('.my-container-class').sortScroll('moveElement',3,-2)//move the element in slot 3 (zero based) 2 slots up (in slot 1)
